*Last updated: June 24, 2026*

## Current State — What Is Built and Deployed

### Core Infrastructure
- Authentication (Supabase Auth)
- Life Graph database schema (PostgreSQL with RLS)
- Document storage (Supabase Storage, user-scoped private buckets)
- Stripe subscription billing (checkout, webhook, lifecycle management)
- Stripe Tax (Tennessee SaaS tax collection, configured)
- Email service (Resend — welcome, confirmation, Life View ready, waitlist confirmation)
- Paywall middleware (proxy.ts — gates all protected routes behind active subscription)
- PWA manifest, service worker (next-pwa), install banner
- App icons (iOS, Android, Windows — all sizes)
- Two founders accounts (thechristinemsmith@gmail.com, founder@krovos.app — active/founder status)

### Life Profile (Onboarding Flow)
- Identity and household (dark brand styling)
- Document upload with smart extraction (26+ document types via Anthropic Claude API)
- Confirm/review (pre-fills from extraction)
- Housing details (pre-filled from mortgage statement)
- Income (pre-filled from pay stub, comma formatting)
- Spending (v4 — pre-fill from Life Graph, CSV extraction, custom line items, debt payments)
- Investments (comma formatting, pre-fill from statements, partner accounts)
- Debt (pre-populated from documents)
- Retirement Goals
- Review (with white-glove prompt wired in — asks one personalized question before Life View generation)

### Core Product Surfaces
- **Life View** — personalized navigation report (6 sections, generated by Claude)
- **Life Compass** — four-zone dashboard (goals, Krovos Says, calendar strip, what's active)
- **Life Calendar** — full monthly grid, timezone-safe, dual-income payday support
- **Krovos Guide** — AI conversation page with life-phase-aware starter questions
- **Krovos Guide API** — full Life Graph context, white-glove recommendation pattern
- **Dashboard** — onboarding completion state, redirects to Life Compass when Life View generated

### Components Built
- DebtLedger, FreedomScenarios, MilestoneEngine, HorizonBuffer
- LeverAnalysis, ProjectionTable, FinancialModes, ChildcareBubble
- KrovosLoader, KrovosSpinner, CurrencyInputFormatted
- KrovosIcon, PWAInstallBanner

### Marketing
- Landing page (centered logo lockup, waitlist form, Life View preview, $15/$19 pricing)
- Subscribe page (founding member pricing, Stripe checkout, trust statement)
- Privacy Policy, Terms of Service pages
- OG image
